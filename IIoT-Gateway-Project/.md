## Project Overview (项目简介)
This project is a full-stack Industrial IoT (IIoT) prototype designed to solve the latency and visibility issues in traditional manufacturing lines. It integrates **Edge Computing**, **Real-time Telemetry**, and **Closed-loop Control**.
本项目是一个全栈工业物联网原型，旨在解决传统产线数据滞后和缺乏可视化的问题。系统集成了边缘计算、实时遥测和闭环控制功能。

## Key Features (核心功能)
*   **Real-time Telemetry (实时遥测)**: Replaced HTTP with **MQTT protocol** to achieve millisecond-level data synchronization.
    (使用 MQTT 协议替代 HTTP，实现毫秒级数据同步。)
*   **Edge Computing (边缘计算)**: Implemented local **State Machine** for signal debouncing and **Closed-loop Control** (Temp -> Relay) to ensure safety.
    (实现本地状态机消抖，以及温度-继电器闭环控制，确保无云端延迟的安全响应。)
*   **Visualization (可视化)**: Dynamic Dashboard built with **Flask + Socket.IO**, pushing real-time data to the browser via WebSockets.
    (基于 Flask 和 Socket.IO 构建动态仪表盘，通过 WebSocket 将实时数据推送到浏览器。)

## Tech Stack (技术栈)
*   **Hardware**: ESP32, OLED, Arcade Button, WS2812 LEDs, Relay, Potentiometer.
*   **Firmware**: C++ / PlatformIO / Arduino Framework.
*   **Backend**: Python Flask / Paho-MQTT / Socket.IO.
*   **Frontend**: HTML5 / CSS3 / JavaScript (WebSocket).

## Hardware Wiring (硬件接线)
| Component | Pin | ESP32 GPIO | Function |
| :--- | :--- | :--- | :--- |
| **Button** | Signal | GPIO 14 | Production Counter (生产计数) |
| **OLED** | SDA/SCL | GPIO 21/22 | Status Display (状态显示) |
| **LED Ring** | DIN | GPIO 27 | Visual Alarm (光效报警) |
| **Potentiometer**| OUT | GPIO 34 | Temp Simulation (温度模拟) |
| **Relay** | IN | GPIO 26 | Fan Control (风扇控制) |

## How to Run (运行指南)

### 1. Firmware (固件)
*   Open `1_Firmware` in VS Code.
*   Configure WiFi & MQTT in `src/main.cpp`.
*   Upload to ESP32.

### 2. Backend (后端)
*   Install dependencies:
    ```bash
    cd 2_Backend
    pip install -r requirements.txt
    ```
*   Start the service:
    ```bash
    python app.py
    ```
*   **Access Dashboard**:
    Open Browser and visit: `http://localhost:5000`
    (打开浏览器访问此地址，查看实时数据看板)
