这是一个基于 **ESP32** 和 **Python Flask** 的全栈物联网演示项目。模拟了工厂产线的**数据采集**、**本地控制**和**Web 实时监控**。

## 📐 系统架构

![System Architecture](3_Docs/architecture.svg)

## ✨ 核心功能

1.  **实时遥测**: 使用 **MQTT 协议** 实现毫秒级数据上报（取代传统的 HTTP 轮询）。
2.  **边缘计算**: 本地实现**闭环控制**（当温度 > 80°C 时自动开启风扇），断网也能安全运行。
3.  **Web 看板**: 基于 Flask + Socket.IO，在浏览器上实时显示产线状态，无需刷新页面。
4.  **产线模拟**:
    *   **按键**: 模拟生产计数。
    *   **电位器**: 模拟设备温度变化。
    *   **灯环**: 红色旋转代表故障，绿色呼吸代表正常。

## 🛠️ 硬件接线 (Wiring)

| 组件 | ESP32 引脚 | 功能描述 |
| :--- | :--- | :--- |
| **按键 (Button)** | GPIO 14 | 生产计数 / 模拟故障 |
| **OLED 屏幕** | GPIO 21 (SDA) / 22 (SCL) | 显示 IP 和 MQTT 状态 |
| **WS2812 灯环** | GPIO 27 | 状态指示灯 |
| **电位器 (Pot)** | GPIO 34 | 模拟温度传感器 |
| **继电器 (Relay)** | GPIO 26 | 风扇控制 (高电平触发) |

## 🚀 如何运行

### 1. 烧录固件
1.  使用 VS Code 打开 `1_Firmware` 文件夹。
2.  修改 `src/main.cpp` 里的 WiFi 和 MQTT 配置。
3.  连接 ESP32，点击左下角的 `Upload` 箭头。

### 2. 启动后端
1.  进入后端目录：
    ```bash
    cd 2_Backend
    ```
2.  安装依赖库：
    ```bash
    pip install -r requirements.txt
    ```
3.  运行程序：
    ```bash
    python app.py
    ```

### 3. 查看效果
打开浏览器访问：`http://localhost:5000`
*   你可以看到实时跳动的产量和温度数据。
*   按下 ESP32 上的按键，网页数据会同步更新。
