这是我的第一个物联网全栈项目！它模拟了一个真实的工厂环境，把数据从 ESP32 发送到网页上显示。

## 实现功能

1.  **按键计数**: 按一下 ESP32 上的按钮，网页上的“产量”就会加 1。
2.  **温度报警**: 转动电位器模拟温度升高，超过 80 度时风扇（继电器）会自动转动降温。
3.  **实时网页**: 不需要刷新页面，数据变化了网页会自动更新（用了 WebSocket 技术）。
4.  **故障模拟**: 长按按钮，机器会进入“故障模式”，网页上会显示红色报警。

## 硬件接线 (怎么连线)

| 硬件名字 | ESP32 引脚 | 作用 |
| :--- | :--- | :--- |
| **按键 (Button)** | GPIO 14 | 增加产量 / 模拟故障 |
| **OLED 屏幕** | GPIO 21 (SDA) / 22 (SCL) | 显示 IP 地址和当前温度 |
| **WS2812 灯环** | GPIO 27 | 状态指示灯 (绿=正常, 红=故障) |
| **电位器 (Pot)** | GPIO 34 | 模拟温度传感器 |
| **继电器 (Relay)** | GPIO 26 | 控制风扇 (高电平触发) |

## 如何运行这个项目

### 第一步：烧录 ESP32
1.  用 VS Code 打开 `1_Firmware` 文件夹。
2.  在 `src/main.cpp` 里填上你的 WiFi 名字和密码。
3.  连接 ESP32，点击左下角的 `Upload` 箭头烧录。

### 第二步：启动电脑端程序
1.  进入后端目录：
    ```bash
    cd 2_Backend
    ```
2.  安装依赖库 (第一次运行需要)：
    ```bash
    pip install -r requirements.txt
    ```
3.  运行程序：
    ```bash
    python app.py
    ```

### 第三步：看效果
打开浏览器访问：`http://localhost:5000`
*   你会看到一个网页，上面显示着当前的产量和温度。
*   操作 ESP32 上的按键，网页数据会同步跳动！

## 数据通信格式

ESP32 会向电脑发送一段 JSON 数据：

*   **`count`**: 当前生产的数量
*   **`temp`**: 当前设备的温度
*   **`status`**: 机器状态 ("running" 正常 / "fault" 故障)

**示例数据**:
```json
{
  "count": 120,
  "temp": 45.5,
  "status": "running"
}
